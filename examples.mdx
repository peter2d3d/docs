# Generate multiple thumbnails from video

```bash
curl --request POST \
  --url http://api.rendi.dev/run-ffmpeg-command \
  --header 'Content-Type: application/json' \
  --header 'X-API-KEY: <api-key>' \
  --data '{ "input_files": 
    { "in_1": 
"https://download.blender.org/peach/bigbuckbunny_movies/big_buck_bunny_720p_stereo.avi"},
    "output_files": {"out_1": "thumbnail1.jpg", "out_2": "thumbnail2.jpg"},
    "ffmpeg_command": "-i {{in_1}} -filter_complex \"[0:v]split=2[first][second];[first]select='\''gte(t,10)'\''[thumb1];[second]select='\''gte(t,20)'\''[thumb2]\" -map \"[thumb1]\" -frames:v 1 {{out_1}} -map \"[thumb2]\" -frames:v 1 {{out_2}}"}'
```

<Accordion title="Get result">
  Running this command will return the command\_id:

  ```json
  {
    "command_id": "7f621856-f42a-4e7e-b2f0-806653564a6b"
  }
  ```

  Poll result:

  ```bash
  curl --request GET \
    --url http://api.rendi.dev/poll-command/7f621856-f42a-4e7e-b2f0-806653564a6b \
    --header 'X-API-KEY: <api-key>'
  ```

  On `SUCCESS`

  ```json
  {
    "output_files": {
      "out_1": "https://storage.rendi.dev/7f621856-f42a-4e7e-b2f0-806653564a6b/thumbnail1.jpg",
      "out_2": "https://storage.rendi.dev/7f621856-f42a-4e7e-b2f0-806653564a6b/thumbnail2.jpg",
    },
    "processing_seconds": 0.88,
    "status": "SUCCESS"
  }
  ```
</Accordion>

# Create several videos with different resolutions

```bash
curl --request POST \
  --url http://api.rendi.dev/run-ffmpeg-command \
  --header 'Content-Type: application/json' \
  --header 'X-API-KEY: <api-key>' \
  --data '{ "input_files": 
    { "in_1": 
"https://download.blender.org/peach/bigbuckbunny_movies/big_buck_bunny_720p_stereo.avi"},
    "output_files": {"out_1": "output_520p.mp4", "out_2": "output_480.mp4"},
    "ffmpeg_command": "-i {{in_1}} -filter_complex \"[0:v]split=2[vid1][vid2];[vid1]scale=trunc(oh*a/2)*2:520:force_original_aspect_ratio=decrease,pad=trunc(oh*a/2)*2:520:-1:-1\"[520p];[vid2]scale=trunc(oh*a/2)*2:480:force_original_aspect_ratio=decrease,pad=trunc(oh*a/2)*2:480:-1:-1\"[480p]\" -map \"[520p]\" -map 0:a -c:a copy -c:v libx264 -preset medium {{out_1}}  -map \"[480p]\" -map 0:a -c:a copy -c:v libx264 -preset medium {{out_2}}"}'
```

<Accordion title="Get result">
  Running this command will return the command\_id:
